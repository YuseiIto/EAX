What is EAX?

EAX or "Easy Apps Exchange"is a web application built by Yusei Ito(@ItyuJ) in Japan. This application has 3 award.(See [Learn more])
This application writes your html apps to selected image files. When you use it,you will get many advantages.

  • Learn More

Advantage to use

  • Local use

    In ios device, You can't open local(file:///) html file. But,in that,You can open local image file.So, EAX converts your HTML to image to make available local use.

  • Online share

    It's difficult to share html file. However, to share image files is very easy. When you use EAX,your html apps will be easy to share because it becomes image file.

  • Use photo library

    Html isn't easy to manage in mobile devices. But , You to manage images is easy because there are photo library.

  • Max 96%
    Compression

    You use EAX, your html will be compressed. In test session, testing text file was compressed 96% in the best case.

  • Encryption

    Use EAX, your html apps will be encrypted. The encription key is the drawing parameter. So you can't open image files without drawing parameter.
    Note:Drawing parameter is publishing by default. Encryption is a optional feature.

  • Reversible

    HTML files converted by EAX is reversible compression and reversible encryption. So you don't need to save original data.
    Attention:If you forget drawing parameter or EAX version, you can't decode your files.

User Experience

EAX's UX is very simple and easy.

The points of UX

  • 10 languages
  • Code highlight
  • 100+ browser available
  • SVG format icons
  • Responsive User Interface

EAX has 5 elements to realize good UX. First is "Multi language available". EAX's language pack is easy to edit because it's written JSON format. Now,EAX available to use 10 language. English and Japanese are translated by Yusei ito. Others are translated by computer software. (Report the translating error:D) 2nd element is "Code highlight". The feature is very powerful. The feature was realized by " EditArea". Thank you "Edit Area" contributors! 3rd element is cross platform. It realized because it built in HTML5,CSS3,and JavaScript. It's good way to unity UX. 4th element is "SVG format icons". That's realized to display as clear in high definition display. 5th element is be Responsive UI design". Responsive UI is modern and beautiful to see. EAX's UX is very simple, that realized by their 5 elements.
